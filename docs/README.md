# Maintained Documentation

Use these documents for current operation and maintenance:

- `../README.md`: mission, install profiles, and primary tool overview.
- `mathdevmcp-colleague-quick-start.md`: colleague installation and first use.
- `mathdevmcp-operator-guide.md`: detailed mathematical workflow operation.
- `mathdevmcp-maintainer-guide.md`: canonical junior-maintainer runbook.
- `mathdevmcp-deployment-guide.md`: optional worker and environment setup.
- `mathdevmcp-support-matrix.md`: supported profiles and version pins.
- `mathdevmcp-versioning-policy.md`: compatibility and deprecation rules.
- `mathdevmcp-security-governance.md`: privacy and claim-boundary rules.
- `mathdevmcp-release-policy.md`: release-profile meanings.

`plans/` and `reviews/` contain historical programs, execution records, and
external feedback. They are evidence for why decisions were made, not current
operating instructions unless a maintained guide links to a specific record.

Generated evidence under `generated/` supports maintained reports. Do not edit
it by hand when a generator exists.
