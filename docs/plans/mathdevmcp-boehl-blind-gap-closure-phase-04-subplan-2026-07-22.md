# Phase 04 Subplan: Source Discovery And Closure

Objective: expose authoritative versus candidate source packages and exact
missing-evidence reasons.

Entry: Phase 03 relations exist.

Artifacts: discovery manifest, provider capability records, closure statuses,
tests, result.

Checks/review: present/absent/failing provider, digest drift, authority labels,
path confinement, no code execution.

Evidence: discovery candidates are not source authority.

Forbidden: automatic authority or execution of discovered code.

Handoff: every unresolved obligation has a reason code and authenticated
evidence is available before promotion.

Stop: provider failure is hidden.
