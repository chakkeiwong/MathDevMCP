# Phase 05 Subplan: Safe Formalization And Validators

Objective: execute bounded algebra/calculus, normalization, timing, ownership,
coefficient, domain, dimension, and arithmetic checks where targets are closed.

Entry: Phase 04 authority/closure records exist.

Artifacts: allowlisted formalization protocol, deterministic traces, paired
tests, result.

Checks/review: exact reviewed-adapter match or abstention; otherwise AST,
depth/symbol/integer/exponent/operation-complexity, CPU/address-space/stack/
wall-time/output, fixed-
worker, identifier, and no-network/file limits; equality/difference,
derivative, branches/poles/nonzero denominators, zero/positive domains,
lag/lead, ownership sets, coefficient mismatch, tolerance, malformed
abstention, no arbitrary execution. Any unresolved branch, pole, denominator,
domain, or conditioning side condition forces abstention.

Evidence: confirmed defects require structured source or an independently
verified transcription, explicit relation, assumptions/domain, tool/version,
and source anchors. Page crops only localize evidence.

Forbidden: silent assumptions, unauthenticated PDF promotion, or CAS-as-
interpretation.

Handoff: direct formalization fixtures meet thresholds without false promotion,
and end-to-end orchestrator tests verify source-packet binding and PDF
abstention. The visible matrix is not a paper-level relation benchmark.

Stop: formalization changes the target or hides uncertainty.
