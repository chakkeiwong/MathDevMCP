# Phase 06 Subplan: Contract And Replay Preparation

Objective: integrate traces, paging, docs, fixture confusion matrix, and a
bounded fresh-agent replay protocol.

Entry: Phases 01--05 pass.

Artifacts: public contract updates, scorecard, frozen prompt, allowed-file and
environment manifests, result.

Checks/review: compact/detail parity, MCP surfaces, hashes, frozen Phase 00
fixture thresholds, source-specific scan, prompt/manifest review.

Evidence: prompt isolation is operational evidence, not scientific evidence.

Forbidden: launching before prompt and implementation freeze.

Handoff: fresh session has a reviewed allowed-file manifest and public CLI;
isolation remains self-reported protocol evidence and is not a holdout claim.

Stop: leakage, schema, or fixture regression.
